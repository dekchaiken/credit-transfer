# โลโก้สำหรับ PDF ใบเทียบโอน

## ต้องการไฟล์ 2 ไฟล์

### 1. `rmutk.png` — ตราประจำมหาวิทยาลัย
- โลโก้วงกลมของ มทร.กรุงเทพ (สีน้ำเงิน-ขาว มีรูปวัด/ตราสัญลักษณ์)
- ขนาดแนะนำ: **300×300 px** ขึ้นไป (สี่เหลี่ยมจัตุรัส)
- รูปแบบ: PNG พื้นหลังโปร่งใส (transparent)

### 2. `ascar.png` — โลโก้ ASCAR
- โลโก้สำนักส่งเสริมวิชาการและงานทะเบียน (ตัว A สีเขียว-ส้ม)
- ขนาดแนะนำ: **400×120 px** (กว้างกว่าสูง)
- รูปแบบ: PNG พื้นหลังโปร่งใส

## วิธีใส่
1. หาไฟล์โลโก้จากเว็บมหาวิทยาลัย หรือเอกสารต้นฉบับ
2. วางไฟล์ในโฟลเดอร์นี้ (`public/logo/`)
3. ตั้งชื่อตามที่ระบุไว้ข้างบนเป๊ะ (`rmutk.png` กับ `ascar.png`)
4. รีสตาร์ท `npm run dev` (ถ้ารันอยู่)
5. กดออก PDF — โลโก้จะปรากฏอัตโนมัติ

## ถ้ายังไม่มีไฟล์
PDF จะเรนเดอร์โดยไม่มีโลโก้ (เว้นที่ว่างไว้) — ระบบจะไม่พัง
